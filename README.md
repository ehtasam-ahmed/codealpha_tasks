# ⚙️ CodeAlpha Mechanical Design & Simulation Projects

This repository contains my project work completed during the CodeAlpha Mechanical Design & Simulation Internship.

## 📌 About the Internship
The internship focuses on developing practical skills in:
- 2D/3D CAD Modeling
- Finite Element Analysis (FEA)
- Motion Simulation
- Real-world Mechanical System Design

The goal is to gain hands-on experience in designing, analyzing, and optimizing mechanical components used in industries like automotive, aerospace, and manufacturing.

---

## 🛠️ Tools & Software Used
- SolidWorks
- ANSYS (for FEA)

---

## 📂 Projects Included

### 🔹 Task 1: 3D CAD Model Creation
Design and modeling of mechanical components with proper dimensions and material selection.

### 🔹 Task 2: Structural Simulation (FEA)
Analysis of stress, deformation, and safety factor using different loading conditions.
